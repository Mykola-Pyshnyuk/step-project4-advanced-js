Воконана робота

    •	Модальні вікна: переписано на ES6-класи, закриваються по Esc і кліку поза формою, усі поля — required.
    •	Форма візиту: динамічно підтягує додаткові поля залежно від лікаря.
    •	Авторизація: токен зберігається після логіну, автоматично завантажуються картки; при виході — список очищується.
    •	cardsPanel.js: повністю оновлено — фільтри, порожній стан, сповіщення, локальне оновлення карток, fallback-картка при помилці.
    •	Редагування: реалізовано модалку редагування картки з оновленням через updateCard.
    •	Visit.js: спрощено створення форм, сабміт через API, fallback при помилці.
    •	VisitCardiologist/Dentist/Therapist: замість спадкування — прості класи для вставки полів.
    •	Request.js: очищено і приведено до нового формату карток.
